# LogoIpsum — Property Listing Website

A multi-page responsive real estate website converted from a Figma design to clean HTML, CSS and JavaScript.

## Live Demo
[View Live Site](https://property-edulovesu.netlify.app/)

## Features
- Responsive on all screen sizes — mobile, tablet, desktop
- Property listing page with clickable house cards
- Property detail page that loads the selected house dynamically
- Contact form with agent profile
- Smooth CSS transitions and hover effects

## Tech Stack
HTML · CSS · Vanilla JavaScript · localStorage API

## Pages
- **index.html** — Main landing page with listings, search and filters
- **property.html** — Property detail page with image gallery and contact form
